# Statistics

Run the statistics script and display the output to the user:

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/stats.py
```

This shows:
- All-time statistics
- Error distribution chart (ASCII bar chart)
- 7-day trend chart
- Most common errors (last 30 days)
- Improvement score
