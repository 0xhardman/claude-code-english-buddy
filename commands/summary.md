# Daily Summary

Run the daily summary script and display the output to the user:

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/daily_summary.py
```

This shows:
- Today's total corrections
- Breakdown by category (spelling, grammar, style, vocabulary)
- Most common mistakes today
- Recent corrections with details
